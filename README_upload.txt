# Toetsweek - Week 2

Upload alle bestanden uit deze map naar de hoofdmap van je GitHub-repository.

## Inhoud
- `index.html` → startpagina van de app
- `manifest.json` → maakt installatie als webapp mogelijk
- `sw.js` → offline cache / service worker
- `subjects.json` → vakkenoverzicht voor de startpagina
- `geschiedenis.html`
- `wiskunde.html`
- `global-perspectives.html`
- `icons/` → app-iconen

## Werking
Open `index.html` in GitHub Pages. Na één keer laden werkt de app ook offline.
